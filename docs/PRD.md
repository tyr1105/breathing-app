# 冰人呼吸法训练应用

一款帮助用户学习和练习冰人呼吸法（Wim Hof Method）的网页应用。

## 项目信息

- **产品经理**：王戬
- **技术负责**：乔乔
- **开始日期**：2026-03-02
- **文档状态**：开发中

## 技术栈

- 前端：React 18 + TypeScript
- 样式：Tailwind CSS
- 构建：Vite
- 动画：Framer Motion
- 状态：Zustand
- 部署：Vercel / Netlify
- 数据：localStorage

## MVP 功能（V1.0）

- [x] 项目初始化
- [ ] 引导式呼吸训练（30次呼吸 × 3轮）
