## 1. 产品概述

一款帮助用户学习和练习冰人呼吸法（Wim Hof Method）的网页应用。通过引导式的呼吸训练、计时器和进度跟踪，帮助用户掌握这种强大的呼吸技巧。

## 2. MVP 范围

✅ **必须实现**：
1. 引导式呼吸训练（完整流程）
   - 30次深呼吸 → 憋气 → 恢复呼吸，共3轮
2. 禅意风格界面设计
   - 深色背景 + 柔和色调
   - 简洁布局，大量留白
3. 呼吸阶段视觉引导
   - 动画圆圈（放大/缩小跟随呼吸节奏）
   - "吸气"/"呼气"文字提示
4. 呼吸次数计数（显示"18/30"进度）
5. 憋气阶段计时器（秒数显示）
6. 恢复呼吸倒计时（15秒）
7. 轮次管理（显示"第2/3轮"）
8. 安全提示弹窗（首次使用）
9. 训练完成总结页面（显示各轮憋气时长）

## 3. 设计风格

禅意风格：深色背景 + 柔和色调 + 大量留白
